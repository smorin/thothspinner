## Overview

## Rich Design Patterns to Follow

### 1. Renderable Protocol Pattern
All Rich components should implement the renderable protocol through `__rich__()` or `__rich_console__()` methods:
- `__rich__()` - Returns a renderable (Text, Table, Group, etc.)
- `__rich_console__()` - For more complex rendering with console access
- `__rich_measure__()` - For width measurement

**Reference**: src/thothspinner/rich/components/base.py:40-43

### 2. Live Display Pattern
Rich's `Live` class manages refresh cycles and updates:
```python
from rich.live import Live
with Live(renderable, refresh_per_second=20) as live:
    live.update(new_renderable)
```

### 3. Composition Pattern
Use Rich's layout primitives for composing components:
- `Group` - Vertical layout
- `Columns` - Horizontal layout  
- `Table` - Grid layout
- `Panel` - Bordered container

### 4. Style Management Pattern
Rich uses `Style` objects for consistent styling:
```python
from rich.style import Style
style = Style(color="#D97706", bold=True)
text = Text("content", style=style)
```

```python
    if kwargs.get('message_text'):

            'text': kwargs['message_text'],
            'shimmer': {'enabled': kwargs.get('message_shimmer', True)}
        }
    if kwargs.get('progress_format'):
        result.setdefault('elements', {})['progress'] = {'format': {'style': kwargs['progress_format']}}
    if kwargs.get('timer_format'):
        result.setdefault('elements', {})['timer'] = {'format': {'style': kwargs['timer_format']}}
    if kwargs.get('hint_text'):
        result.setdefault('elements', {})['hint'] = {'text': kwargs['hint_text']}

    valid_types = set(self._render_order)
    for component_type in result.get('elements', {}):
        if component_type not in valid_types:
            raise KeyError(f"Invalid component type: {component_type}. Valid types: {valid_types}")
    
    return result

def _resolve_config(self, component_type: str, state: Optional[ComponentState] = None) -> Dict:
    """Resolve configuration with proper inheritance: component > state > global.
    
    Raises KeyError for invalid component types.
    """
    if component_type not in self._render_order:
        raise KeyError(f"Invalid component type: {component_type}")
    
    config = self.config['defaults'].copy()
    
    # Apply state-specific config if provided
    if state:
        state_name = state.name.lower()
        if state_name in self.config.get('states', {}):
            config.update(self.config['states'][state_name].get(component_type, {}))
    
    # Apply component-specific config
    if component_type in self.config.get('elements', {}):
        config.update(self.config['elements'][component_type])
    
    return config
```

```python
def _create_all_components(self) -> None:
    """Eagerly create all 5 components on initialization.
    
    Like Rich's Status which creates its Spinner immediately.
    """
    from .components import (
        SpinnerComponent, MessageComponent, ProgressComponent,
        TimerComponent, HintComponent
    )
    
    with self._lock:
        # Create all 5 components with resolved configs (eager creation)
        self._components = {
            "spinner": SpinnerComponent(**self._resolve_config("spinner")),
            "message": MessageComponent(**self._resolve_config("message")),
            "progress": ProgressComponent(**self._resolve_config("progress")),
            "timer": TimerComponent(**self._resolve_config("timer")),
            "hint": HintComponent(**self._resolve_config("hint"))
        }
        
        # Validate render order matches actual components
        if "render_order" in self.config:
            order = self.config["render_order"]
            invalid = set(order) - set(self._components.keys())
            if invalid:
                raise KeyError(f"Invalid components in render_order: {invalid}")
            # Convert to immutable tuple
            self._render_order = tuple(order)

def get_component(self, component_type: str) -> Any:
    """Type-based component access with validation.
    
    Raises:
        KeyError: If component_type is invalid
    """
    with self._lock:
        if component_type not in self._components:
            raise KeyError(
                f"Invalid component type: {component_type}. "
                f"Valid types: {list(self._components.keys())}"
            )
        return self._components[component_type]
```

**Why**: Fixed component set eliminates dynamic creation complexity. Type-based access is intuitive since we have exactly one of each type.

```python
def __rich_console__(self, console: Console, 
                     options: ConsoleOptions) -> RenderResult:
    """Advanced rendering with console access for width constraints.
    
    Thread-safe rendering with proper locking.
    """
    with self._lock:
        # Filter visible components using immutable render_order
        visible = [
            self._components[name] 
            for name in self._render_order
            if name in self._components and self._components[name].visible
        ]
        
        if not visible:
            yield Text("")
            return
        
        # Handle fade-away animation
        if hasattr(self, '_fade_progress') and self._fade_progress is not None:
            visible = self._apply_fade_away(visible)
        
        # Use Columns for horizontal layout with proper spacing
        if self.config.get("layout", "horizontal") == "vertical":
            # Vertical layout using Group
            from rich.group import Group
            yield Group(*visible)
        else:
            # Horizontal layout (default) using Columns
            from rich.columns import Columns
            yield Columns(visible, padding=(0, 1), expand=False)

def __rich_measure__(self, console: Console, 
                     options: ConsoleOptions) -> Measurement:
    """Measure total width needed for proper layout.
    
    Implements Rich protocol for width calculation.
    """
    from rich.measure import Measurement
    
    with self._lock:
        visible = [
            self._components[name]
            for name in self._render_order
            if name in self._components and self._components[name].visible
        ]
        
        if not visible:
            return Measurement(0, 0)
        
        # Calculate total width with padding
        measurements = [Measurement.get(console, options, c) for c in visible]
        padding_width = (len(visible) - 1) * 2  # padding between components
        
        min_width = sum(m.minimum for m in measurements) + padding_width
        max_width = sum(m.maximum for m in measurements) + padding_width
        
        return Measurement(min_width, max_width)
```

```python
import threading
import time

def start(self) -> None:
    """Begin in in_progress state."""
    with self._lock:
        self._state = ComponentState.IN_PROGRESS
        self._start_time = time.time()
        
        # Cancel any pending auto-clear
        if self._clear_timer:
            self._clear_timer.cancel()
            self._clear_timer = None
        
        for name in self._render_order:
            component = self._components[name]
            if hasattr(component, 'start'):
                component.start()

def success(self, message: Optional[str] = None, 
           duration: Optional[float] = None) -> None:
    """Transition to success state with optional auto-clear.
    
    Args:
        message: Optional success message
        duration: Override default success_duration for auto-clear
    """
    with self._lock:
        self._validate_transition(ComponentState.SUCCESS)
        self._state = ComponentState.SUCCESS
        
        # Propagate to components
        for name in self._render_order:
            component = self._components[name]
            if hasattr(component, 'success'):
                component.success(message)
        
        # Handle auto-clear with threading.Timer
        clear_duration = duration or self.success_duration
        if clear_duration:
            self._clear_timer = threading.Timer(clear_duration, self.clear)
            self._clear_timer.start()

def error(self, message: Optional[str] = None,
         duration: Optional[float] = None) -> None:
    """Transition to error state with optional auto-clear.
    
    Args:
        message: Optional error message  
        duration: Override default error_duration for auto-clear
    """
    with self._lock:
        self._validate_transition(ComponentState.ERROR)
        self._state = ComponentState.ERROR
        
        # Propagate to components
        for name in self._render_order:
            component = self._components[name]
            if hasattr(component, 'error'):
                component.error(message)
        
        # Handle auto-clear with threading.Timer
        clear_duration = duration or self.error_duration
        if clear_duration:
            self._clear_timer = threading.Timer(clear_duration, self.clear)
            self._clear_timer.start()

def reset(self) -> None:
    """Return to in_progress state."""
    with self._lock:
        self._state = ComponentState.IN_PROGRESS
        for name in self._render_order:
            component = self._components[name]
            if hasattr(component, 'reset'):
                component.reset()

def clear(self) -> None:
    """Stop and clear display."""
    with self._lock:
        # Cancel any pending auto-clear
        if self._clear_timer:
            self._clear_timer.cancel()
            self._clear_timer = None
        
        # Clear all components
        for name in self._render_order:
            component = self._components[name]
            component.visible = False

def stop(self) -> None:
    """Alias for clear() to match Rich's Status API."""
    self.clear()
```

**Why**: Methods follow command pattern for state changes. Duck typing with `hasattr` allows flexible component integration.

```python
def _propagate_state(self, state: ComponentState, 
                     message: Optional[str] = None) -> None:
    """Propagate state changes to all components.
    
    Thread-safe propagation with proper locking.
    """
    # Must be called with lock held
    state_name = state.name.lower()
    state_config = self._resolve_config(state_name, state)
    behavior = state_config.get("behavior", "indicator")
    
    for name in self._render_order:
        component = self._components[name]
        component_config = self._resolve_config(name, state)
        
        # Apply state-specific configuration
        if behavior == "disappear":
            component.visible = False
        elif behavior == "indicator":
            # Call component's state method if it exists
            method_name = state_name  # 'success', 'error', etc.
            if hasattr(component, method_name):
                getattr(component, method_name)(message)
        elif behavior == "message":
            if hasattr(component, 'set_text'):
                component.set_text(message or state_config.get("message"))
        elif behavior == "both":
            # Both indicator and message
            method_name = state_name
            if hasattr(component, method_name):
                getattr(component, method_name)(message)
            if hasattr(component, 'set_text'):
                component.set_text(message or state_config.get("message"))
        
        # Apply color changes from state config
        if "color" in component_config and hasattr(component, 'color'):
            component.color = component_config["color"]
```

**Why**: Duck typing with `hasattr`/`getattr` provides flexible component integration without tight coupling.

```python
def _apply_fade_away(self, components: List) -> List:
    """Apply fade-away animation to components."""
    
    fade_config = self.config.get("fade_away", {})
    if not fade_config.get("enabled", False):
        return []
    
    direction = fade_config.get("direction", "left-to-right")
    interval = fade_config.get("interval", 0.05)
    
    elapsed = time.time() - self._fade_start_time
    faded_count = int(elapsed / interval)
    
    if direction == "left-to-right":
        return components[faded_count:]
    else:  # right-to-left
        return components[:-faded_count] if faded_count < len(components) else []

def _start_fade_away(self) -> None:
    """Initialize fade-away animation."""
    self._fade_start_time = time.time()
    self._fade_progress = 0
```

**Why**: Time-based animation using elapsed time works with Rich's Live refresh cycle. No internal timers needed.

```python
    @lru_cache(maxsize=128)
    def _build_layout(self, component_ids: tuple) -> Columns | Group:
        """Cache layout construction for unchanged components."""
        visible = [c for c in self.components if id(c) in component_ids]
        if self.config.get("column_layout", False):
            return Group(*visible)
        return Columns(visible, padding=(0, 1))
    
    def __rich__(self) -> Any:
        """Optimized rendering with caching."""
        current_time = time.time()
        
        # Use cached render if recent
        if (self._render_cache and 
            current_time - self._cache_time < self._cache_duration):
            return self._render_cache
        
        # Build new render
        visible_ids = tuple(id(c) for c in self.components if c.visible)
        layout = self._build_layout(visible_ids)
        
        self._render_cache = layout
        self._cache_time = current_time
        return layout
```

**Why**: Caching reduces CPU usage. LRU cache for layout construction avoids rebuilding identical layouts.

**Target**: < 10% CPU at 20 FPS per prd_v1.md:611.

## Rich Update Pattern

Following Rich's spinner.update() pattern from /Users/stevemorin/c/rich/rich/spinner.py:95-114:

```python
def update(
    self,
    *,
    text: Optional[str] = None,
    style: Optional[StyleType] = None,
    speed: Optional[float] = None,
) -> None:
    """Update component properties.
    
    All parameters are keyword-only to ensure explicit intent.
    Only non-None values cause updates.
    """
    if text is not None:
        self.text = Text.from_markup(text) if isinstance(text, str) else text
    if style is not None:
        self.style = style
    if speed is not None:
        self._update_speed = speed  # Applied on next render
```

This pattern:
1. Uses keyword-only arguments (after `*`)
2. Only updates when values are not None
3. Defers some updates to next render cycle for smoothness
4. Converts strings to Rich Text objects as needed

- Fixed set of 5 components (no dynamic creation)
